# nyt_cli
this is a very simple cli that fetches the top 10 most popular NYT articles and prints them in a simple and easy to view layout.

# running
1. go to the [releases tab](https://github.com/DaBlower/nyt_cli/tags) and download the latest version for your architecture and os.
2. extract the archive
3. create a .env file and add your NYT api key, which you can get from [here](https://developer.nytimes.com/) (use .env.example as a reference)
4. now you can run it!
